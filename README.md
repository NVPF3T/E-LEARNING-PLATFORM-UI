# DESCRIPTION:
NextEdTech – Modern E-Learning Platform (UI)

NextEdTech is a modern, responsive e-learning web application built using HTML, CSS, and JavaScript. This project represents a real-world online learning platform where users can explore courses, track their progress, watch video lessons, and manage their profile through a clean and interactive user interface.

The Home page introduces the platform with a visually engaging hero section, key platform statistics, featured courses, learning paths, student testimonials, and a strong call-to-action. It is designed to clearly communicate the platform’s value and guide users toward starting their learning journey.

The Dashboard serves as a personalized learning center. It displays overall course completion using circular progress indicators, weekly learning activity through bar charts, and enrolled course cards. This section focuses on motivation and consistency by showing learning streaks, total learning hours, and progress percentage.

The Course Player page provides a structured video-based learning experience. It includes an embedded video player with lesson controls and a sidebar listing course lessons. Users can switch between lessons, and the active lesson updates dynamically, closely simulating a real online course platform. This section highlights the platform’s focus on hands-on, video-driven learning.

The Profile page allows users to manage their personal and learning information. It features profile details, badges, learning statistics, editable account settings, and basic security options. Tab-based navigation keeps the interface organized and easy to use.

Key Features

Responsive and modern UI design

Multi-page navigation (Home, Dashboard, Course Player, Profile)

Video-based course player with lesson switching

Learning progress tracking and activity visualization

Interactive dashboard with charts and progress indicators

Profile management with editable user details

Clean layout with reusable components and consistent styling
